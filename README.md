# 🏡 California House Price Prediction using Linear Regression

## 📌 Overview

This project predicts California house prices using the **Linear Regression** algorithm from Scikit-learn. It uses the California Housing dataset available in Scikit-learn and demonstrates the complete machine learning workflow, including data loading, preprocessing, model training, prediction, and evaluation.

---

## 🎯 Objective

Build a regression model capable of predicting house prices based on housing-related features.

---

## 📂 Dataset

**California Housing Dataset**

Source:
Scikit-learn (`fetch_california_housing()`)

Features include:

- Median Income
- House Age
- Average Rooms
- Average Bedrooms
- Population
- Average Occupancy
- Latitude
- Longitude

Target:

- Median House Value

---

## 🛠 Technologies Used

- Python
- Pandas
- Scikit-learn
- Matplotlib

---

## 🤖 Machine Learning Algorithm

- Linear Regression

---

## 📊 Workflow

1. Load Dataset
2. Convert to DataFrame
3. Train-Test Split
4. Train Linear Regression Model
5. Predict House Prices
6. Evaluate Model
7. Visualize Predictions

---

## 📈 Evaluation Metrics

- Mean Absolute Error (MAE)
- Mean Squared Error (MSE)
- R² Score
